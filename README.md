# cicd-monitoring
Collection of cicd pipelines.
How to install the monitoring tool.

## AWS
```bash
└── aws
    ├── argocd
    ├── grafana
    ├── loki
    └── prometheus
```

## GCP
```bash
└── gcp
    ├── argocd
    ├── grafana
    ├── loki
    └── prometheus
```

## Onpremise
```bash
└── onpremise
    ├── argocd
    ├── jenkins
    └── metallb
```

## gitlab-cicd
```bash
└── gitlab-cicd
```

## gitlab-runner
```bash
└── gitlab-runner
```

## github-cicd
```bash
└── github-cicd
    ├── aws
    └── gcp
```
